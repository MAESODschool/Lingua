3. Regular Verbs
--------------------------------------------------
Do not present all rules in one block.
Each rule must be taught separately.
For each rule, use this micro-structure:
1. teacher intro
2. one idea
3. example cards
4. one small interactive question
5. teacher response
6. one quick drill
7. then continue
Rule groups:
Rule 1:
General verbs add -ed
Examples:
walk → walked
jump → jumped
wash → washed
look → looked
Rule 2:
Ending in e add only -d
Examples:
love → loved
like → liked
dance → danced
close → closed
Rule 3:
Ending in y
Case A: vowel + y
play → played
enjoy → enjoyed
Case B: consonant + y
study → studied
cry → cried
try → tried
Rule 4:
Short CVC verbs double final consonant + ed
stop → stopped
plan → planned
drop → dropped
clap → clapped
For each rule:
Do not show all examples as a paragraph.
Show them as example cards or thought bubbles.
Teacher example:
“คำนี้ลงท้ายด้วย e อยู่แล้ว
เพราะฉะนั้นเราไม่เติม e ซ้ำ
เราเติมแค่ -d”
Then show:
love → loved
like → liked
Then ask:
“ถ้า dance เป็นอดีต เธอคิดว่าควรเป็นข้อใด?”
--------------------------------------------------
4. Irregular Verbs
--------------------------------------------------
Also teach step-by-step.
Explain:
“Irregular verbs ไม่เดินตามกฎการเติม -ed
มันเปลี่ยนรูปไปเอง
ดังนั้นเราต้องจำเป็นคำ ๆ”
Compare:
walk → walked
play → played
study → studied
with:
go → went
eat → ate
see → saw
buy → bought
Use short dialogue, example cards, one or two interactions, then drills.
